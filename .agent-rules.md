# Project Rules for AI Agent

- Follow the existing code style and architecture of the project.
- Use TypeScript best practices if applicable.
- Prefer modern ES6+ syntax.

## Documentation Rule
- Always add JSDoc comments to new or undocumented functions. Ensure parameters and return types are clearly described.


## Database D1 Updates
- Any schema changes to the local db should also be pushed to the remote immediately - local and remote environments should be fully up to date and in sync 

## Error Handling
- Provide proper error feedback, implement toaster

## Security Check
- After completing the task, check if any credentials are leaked out in any of the pages or git specific pages
